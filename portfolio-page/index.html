Projects

Grubbernaut

Grubbernaut is an app that puts hard to access information into the hands of users in an easy to use and fun format.

Grubbernaut takes health inspection score data compiled through three Wake County API's and allows the user to compare the scores to Yelp ratings which are gleaned from a fourth API-- Yelp Fusion API-- in one quick step. Grubbernaut gives a user more decision-making power when choosing where to dine.

This app was developed using JavaScript, Django, HTML and CSS. Graphic elements were created using Chart.js. This project was created over the course of two weeks with three classmates at Momentum Learning Immersive Coding Program.
